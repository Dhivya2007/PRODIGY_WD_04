# 🌐 Personal Portfolio Website

## 👩‍💻 About the Project
This is my personal portfolio website built using **HTML, CSS, JavaScript, and Bootstrap**.  
It showcases my skills, projects, education, and experience as a Computer Science Engineering student.

---

## 🚀 Features

- Responsive modern UI design
- Dark theme with glassmorphism effects
- Smooth navigation bar
- Skills section with progress bars
- Project showcase cards
- About and Experience sections
- Contact section with Email, GitHub, and LinkedIn links

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

- Technical Skills
- Soft Skills
